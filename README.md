# Gestor de tareas en python

Proyecto personal desarrollado en Python para practicar el uso y la manipulacion de listas en Python.

El programa permite registrar:
- Agregar tarea
- Marcar tarea como completada
- Eliminar tarea
- Ver todas las tareas
- Salir del programa

El sistema muestra un resumen del día y permite reiniciar sesiones.

### Conceptos aplicados:
- Uso de listas en Python
- Control de flujo con if, elif, else
- Bucles while True
- Manejo de errores con try/except
- Uso de enumerate()
- Validación de datos ingresados por el usuario
- Pensamiento lógico y estructura de programas interactivos
